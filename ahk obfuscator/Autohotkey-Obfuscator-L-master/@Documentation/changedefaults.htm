Change Obfuscation Encoding Defaults for Autohotkey Object Types

In order to change the obfuscation encoding defaults for functions, you use the CHANGE_DEFAULTS command.

For an explanation of the meaning of each individual obfuscation parameter, refer to Understanding the Standard Parameter Set.

You can also use the RESTORE_DEFAULTS command to restore what they were before the change. Only one restore step back is allowed.

Change Function Defaults

;$OBFUSCATOR: $FUNCS_CHANGE_DEFAULTS: {minobfstr}, {maxobfstr}, {obfrows}, {fragsperrow}, {minfragstr}, {maxfragstr} 
;$OBFUSCATOR: $FUNCS_RESTORE_DEFAULTS:

Change Parameter Defaults

;$OBFUSCATOR: $PARAMS_CHANGE_DEFAULTS: {minobfstr}, {maxobfstr}, {obfrows}, {fragsperrow}, {minfragstr}, {maxfragstr} 
;$OBFUSCATOR: $PARAMS_RESTORE_DEFAULTS:
